Abstract

Este trabalho tem por base a ontologia desenvolvida na aula, tendo como fonte de informação a folha de notas da UC de PRI contendo todas as notas e informaçoes relativas a cada aluno.

O primeiro passo foi implementar à mão uma versão básica no protégé de modo a conseguir obter um exemplo de como a estrutura seria na versão textual.

Uma vez que a lista de alunos é algo extensiva, segui para a implmentação de um parser em python que retirava a informação da tabela das notas e a colocava num ficheiro À parte já com o formato desejado.

Para finaçizar é só copiar a informação contida no ficheiro à parte e colar no ficheiro principal.
