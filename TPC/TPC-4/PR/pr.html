Abstract

Para este trabalho foi implementada uma aplicação em NodeJS para navegar com querys em SPARQL num repositório do graphDB sem que seja necessario especificar querys. Para isto foram desenvolvidas duas paginas em que, na primeira é apresentada a lista de todos os Arquitetos na base de dados com um botão que nos leva para a página individual. Na página individual encontramos uma tabela com grande parte da informação do advogado selecionado. Fiz este trabalho com o repositório dos advogados porque o do arquivo musical nao estava a responder corretamente aos meus pedidos.
